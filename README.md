Olá! Meu nome é **Gabriel** (17 anos). Comecei a programar como um hobby, fazia somente bots para WhatsApp, Discord e Telegram. Mas depois eu comecei a fazer sites e aplicativos, e hoje trabalho com desenvolvimento fullstack, criando soluções completas para web e mobile.

Sempre fui apaixonado por tecnologia e automação, e hoje transformo essa paixão em projetos reais que ajudam pessoas e empresas.
